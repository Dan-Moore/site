# Arch Linux with Hyprland
[Hyprland - Arch Wiki](https://wiki.archlinux.org/title/Hyprland) is a popular tiling manager that's easy to get started
with by sharing dot files.  Check out [Hyprland's Hall of Fame](https://hyprland.org/hall_of_fame/) for show cases of hyprland dot files.


[KooL's Arch-Hyprland](https://github.com/JaKooLit/Arch-Hyprland) has a public dot file you can test out Hyprland.
Running this command will configure and install hyprland for an arch machine.
```sh
git clone --depth=1 https://github.com/JaKooLit/Arch-Hyprland.git ~/Arch-Hyprland
cd ~/Arch-Hyprland
chmod +x install.sh
./install.sh
```

## Waybar Config
The Waybar is the menu bar of Hyprland, you can edit the config with this command.
```sh
cd ~/.config/waybar
nano config
```
